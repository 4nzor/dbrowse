# Documentation

This directory contains additional documentation for dbrowse.

## Contents

- `SCREENSHOTS.md` - Guide for adding screenshots to the project
- `screenshots/` - Directory for storing screenshots (add your screenshots here!)

## Contributing Documentation

If you want to improve the documentation:

1. Take screenshots of dbrowse in action
2. Add them to `screenshots/` directory
3. Update `README.md` in the root to reference the screenshots
4. Submit a PR!
