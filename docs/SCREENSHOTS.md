# Screenshots Guide

This directory is for storing screenshots of dbrowse in action.

## How to Add Screenshots

1. Take a screenshot of dbrowse showing:
   - Main interface with three columns
   - Table browsing
   - Filtering/Sorting
   - Data export
   - Any other interesting features

2. Save screenshots as PNG files with descriptive names:
   - `main-view.png` - Main interface
   - `table-browsing.png` - Browsing tables
   - `filtering.png` - Using WHERE/ORDER BY
   - `export.png` - Exporting data

3. Add them to this directory and update README.md with image references

## Screenshot Tools

- **macOS**: `Cmd + Shift + 4` (select area) or `Cmd + Shift + 3` (full screen)
- **Linux**: Use `gnome-screenshot`, `scrot`, or `flameshot`
- **Windows**: `Win + Shift + S` (Snipping Tool)

## Tips

- Use a terminal with good color scheme
- Show interesting data (not empty tables)
- Include examples of filtering/sorting
- Show the status messages at the bottom
